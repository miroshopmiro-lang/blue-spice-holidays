# Blue Spice Holidays

Premium travel website for **Blue Spice Holidays** - *Warmth & Luxury across every journey.*

Built with Vite + React (SWC), Tailwind CSS, GSAP, Framer Motion, and Lenis smooth scroll. The landing page recreates the supplied reference design (Hero -> Why -> Destinations -> Service Tiers -> Bento -> How to Book -> Footer); `/packages`, `/gallery`, and `/contact` reuse the same design system.

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is written to `dist/`. Deploys to Netlify via `netlify.toml` with SPA redirects.

## Content source

All copy, destinations, services, and testimonials come from `blue_spice_holidays_info.md`.

## Asset notes

Destination/gallery imagery uses Unsplash placeholders. Replace with licensed Blue Spice photography before launch, and drop the hero video at `public/Blue spice holidays hero.mp4` and logo at `public/images/logo.jpeg`.
